WEB DESIGNS AND DYNAMIC AND STATIC WEB PAGES

Q1:  Define static and dynamic web pages. What are the key differences in terms of functionality, user interaction, and the technologies used to create them? Provide examples of scenarios where each type would be appropriate to use.

STATIC WEB PAGES

  • Functionality: Limited to displaying fixed content

  • User Interaction: Minimal or no user interaction.

  • Technologies: Primarily HTML and CSS.

  • Examples: Personal websites, simple landing pages, online brochures.

DYNAMIC WEB PAGES

  • Functionality:Can display dynamic content, interact with databases, and respond to user input.

  • User Interaction:High level of user interaction, allowing users to customize content, make purchases, or participate in online communities.

  • Technologies:HTML, CSS, JavaScript, server-side scripting languages (PHP, Python, Ruby on Rails, etc.), and databases.

  • Examples:E-commerce websites, social media platforms, online banking portals, web applications.

RESPONSIVE WEB DESIGNS

  • Definition:A design approach that makes websites adaptable to different screen sizes and devices (desktop, tablet, mobile).

  • Importance:Ensures a consistent user experience across various devices, improves search engine rankings, and enhances user engagement.

CSS MEDIA QUERIES AND FLEXIBLE LAYOUTS

  • CSS Media Queries:Allow you to apply specific styles based on the device's screen size, orientation, and other characteristics.

  • Flexible Layouts:Employ techniques like fluid grids and flexible images to adjust the layout and content to fit different screen sizes.
